## 3. 合并内容

本次合并将未合并版本中与购物车 UI 展示相关的后端改进合并到当前项目：

### 3.1 Cart.java

新增两个非持久化字段，用于向前端购物车页面返回商品名称与单价：

```java
@TableField(exist = false)
private String productName;

@TableField(exist = false)
private BigDecimal productPrice;
```

### 3.2 CartRepository.java

新增两个方法，用于处理逻辑删除的购物车记录，避免重复加购时的唯一键冲突：

```java
@Select("SELECT * FROM cart WHERE user_id = #{userId} AND product_id = #{productId} LIMIT 1")
Optional<Cart> findIncludingDeleted(Long userId, Long productId);

@Update("UPDATE cart SET quantity = #{quantity}, deleted = 0 WHERE id = #{id}")
int restoreDeleted(@Param("id") Long id, @Param("quantity") int quantity);
```

### 3.3 CartServiceImpl.java

- 在 `addItem` 中，当正常查询不到购物车时，检查是否存在被软删除的记录，若存在则恢复并更新数量；
- 在 `listCartItems` 中，根据 `productId` 查询商品信息并填充 `productName` 与 `productPrice`，使前端购物车可直接展示商品名称与价格。

## 6. UI 兼容性检查

### 6.1 浏览器兼容性

项目基于 **Vue 3.4 + Element Plus 2.7**，官方支持：

- Chrome / Edge（最新 2 个版本）
- Firefox（最新 2 个版本）
- Safari（最新 2 个版本）

### 6.2 Playwright 配置

当前 `playwright.config.cjs` 仅配置了 `chromium` 桌面浏览器，建议后续扩展：

- 增加 `firefox` 与 `webkit` 项目；
- 增加移动设备视口配置（如 `iPhone SE`、`Pixel 5`）。

### 6.3 响应式设计评估

- 页面布局采用 `max-width` + `margin: 0 auto`，具备基础适配能力；
- `Home.vue` 使用 `el-row/el-col` 栅格，但仅使用固定 `span`，未配置 `xs/sm/md/lg/xl` 断点，小屏设备可能出现列挤压；
- `Cart.vue` 使用 `el-table`，在窄屏设备下需要横向滚动；
- 建议在后续迭代中为关键页面补充响应式断点，优化移动端体验。
