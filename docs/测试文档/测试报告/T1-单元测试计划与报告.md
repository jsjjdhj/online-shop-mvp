## 一、T1 开发任务说明

### 1.1 任务范围

| 子任务 | 说明 | 涉及文件 |
|--------|------|----------|
| 用户注册 | 唯一用户名 + 合规密码 + 密码确认 | UserServiceImpl, UserController |
| 用户登录 | 密码校验 + JWT Token 生成 | UserServiceImpl, UserController |
| 登录安全 | 连续 5 次失败锁定账号，锁定到期自动解锁 | UserServiceImpl |
| 购物车添加 | 商品加购，支持数量累加，原子库存校验 | CartServiceImpl, CartController |
| 购物车更新 | 修改数量，数量为 0 时自动删除 | CartServiceImpl |
| 购物车删除 | 单条记录删除 | CartServiceImpl |
| 购物车查询 | 列出当前用户全部购物车商品 | CartServiceImpl |

### 1.2 用户状态定义

| 状态码 | 名称 | 说明 |
|--------|------|------|
| 0 | 正常 | 可正常登录 |
| 1 | 已锁定 | 连续 5 次登录失败后锁定 |

### 1.3 购物车数量规则

- 单次添加数量范围：1 ~ 99
- 重复添加同一商品时数量累加，累加后总数量不得超过 99
- 添加/更新时校验商品库存，库存不足则抛出 BusinessException
