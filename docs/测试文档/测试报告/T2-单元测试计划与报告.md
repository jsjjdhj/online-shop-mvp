### 3.3 E2E 端到端测试脚本

端到端脚本（`tests/e2e/example.spec.js`）包含以下自动化步骤：

```
1. 打开登录页 → 输入用户名/密码 → 点击登录
2. 跳转首页 → 搜索商品 → 点击商品卡片
3. 加入购物车 → 验证成功提示
4. 进入购物车 → 点击去结算
5. 填写收货信息（姓名/电话/地址）
6. 提交订单 → 验证订单提交成功
```
