## 一、T3 开发任务说明

### 1.1 任务范围

| 子任务 | 说明 | 涉及文件 |
|--------|------|----------|
| 商品管理 | 商品新增、编辑、下架 | AdminController, ProductServiceImpl |
| 订单管理 | 管理员确认订单、发货操作 | AdminController, OrderServiceImpl |
| 订单状态流转 | 完整流程含取消与库存回补 | OrderServiceImpl |
| 管理员订单列表 | 分页查询全部订单，支持按状态筛选 | OrderServiceImpl.listAllOrders() |
| 管理员订单详情 | 查看订单基本信息 + 商品明细 | OrderServiceImpl.getAdminOrderDetail() |
| 用户确认收货 | 已发货 → 已完成 | OrderServiceImpl.completeOrder() |

### 1.2 订单状态定义

| 状态码 | 名称 | 说明 |
|--------|------|------|
| 0 | 待确认 | 用户下单后初始状态 |
| 1 | 已确认 | 管理员确认订单 |
| 2 | 已付款 | 用户模拟付款 |
| 3 | 已发货 | 管理员发货 |
| 4 | 已完成 | 用户确认收货 |
| 5 | 已取消 | 用户取消（自动恢复库存） |

### 1.3 状态流转

待确认(0) → 已确认(1) → 已付款(2) → 已发货(3) → 已完成(4)
   ↓            ↓
  取消(5)     取消(5)
