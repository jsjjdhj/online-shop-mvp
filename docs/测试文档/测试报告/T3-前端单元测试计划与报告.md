## 一、测试目标

为 T3 管理员端前端功能补充全面的单元测试，覆盖核心功能模块、组件和业务逻辑：

### 4.3 覆盖的测试场景

**API 层（16 个）**：
- 所有 HTTP 方法：GET / POST / PUT / DELETE
- 所有 API 端点：商品 CRUD + 订单 CRUD + 管理员操作

**管理员视图层（28 个）**：
- 页面渲染：标题、布局、子组件

- 表单填充：初始值、编辑回填、重置清空
- 错误处理：API 失败不导致崩溃
- 边界条件：缺少可选字段的容错
